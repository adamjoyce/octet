L System Project Readme - Adam Joyce

I have included Andy Thomason's entire octet framework in the lsystemproject.zip folder.

My L System dictory is located at the following path:

...\octet\src\examples\l_systems

In this folder you will find:

- The three core code files:
   - l_system_generator.h
   - l_systems.h
   - main.cpp

- The Visual Studio files/folders:
   - l_systems.sln
   - l_systems.vcxproj
   - l_systems.vcsproj.filters
   - l_systems.xcodeproj (for xcode projects)

- The eight different CSV files containing each of the L-system parameters:
   - data1.csv
   - data2.csv
   - data3.csv
   - data4.csv
   - data5.csv
   - data6.csv
   - data7.csv
   - data8.csv

- The two report files in different formats:
   - adamjoyce-L-system-2015.docx
